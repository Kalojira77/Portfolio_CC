# Manuel utilisateur

## Objet du document
Ce guide explique comment gérer le contenu du site WordPress :
- modifier les pages
- gérer les fiches films
- comprendre le formulaire de contact

---

## Modifier une page

1. Se connecter à WordPress
2. Aller dans **Pages → Toutes les pages**
3. Cliquer sur la page souhaitée
4. Modifier le texte
5. Cliquer sur **Mettre à jour**

Les changements sont visibles immédiatement.

---

## Modifier la photo sur une page profil

1. Cliquer sur la photo
2. Cliquer sur **Remplacer**
3. Choisir une nouvelle image
4. Vérifier l’aperçu
5. Cliquer sur **Mettre à jour**

Éviter les images trop lourdes.

---

## Gérer les films

Les films se trouvent dans :  
**Films → Tous les films**

### Ajouter un film

1. **Films → Ajouter**
2. Saisir :
   - Titre
   - Texte descriptif
   - Image mise en avant
3. Renseigner les champs **facultatifs** de la fiche technique :
   - année, durée, réalisation, montage, etc.
4. Cliquer sur **Publier**

### Modifier un film

1. Ouvrir le film
2. Modifier les champs
3. Cliquer sur **Mettre à jour**

### Vidéo Vimeo

Si une URL Vimeo est saisie dans le champ prévu :
✔ la vidéo apparaît automatiquement en bas de la fiche film.

Si le champ est vide :
✔ rien ne s’affiche.

---

## Formulaire de contact

La page Contact contient un formulaire.

En cas de problème :
- vérifier le dossier spam
- contacter l’administratrice du site

---

## Conseils d’utilisation

- privilégier des textes clairs et courts
- relire avant publication
- utiliser des images légères
- ne pas modifier les réglages techniques
- sauvegarder le travail régulièrement

---

## Assistance

Si un problème survient :
1. noter la page concernée
2. décrire ce qui se passe
3. joindre une capture d’écran si possible
